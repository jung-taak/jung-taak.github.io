VISKY is a mobile robot developed for autonomous exploration in unknown, unstructured, and hazardous environments, as part of the Challengeable Future Defense Technology Research and Development Program funded by the Agency for Defense Development (ADD). The system is built on an end-to-end visual navigation model, combining multi-modal localization with real-time safe path replanning to support robust long-horizon navigation in the field.

My work on this project focuses on training the end-to-end path prediction model and the safety guardrail model that keeps navigation safe over long-horizon missions.
